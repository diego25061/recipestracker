# Recipes Tracker

Simple portfolio project to showcase the development of a complete ReactJS app with a simulated backend.
Deployed in a DigitalOcean VPS using express js + pm2

[Test here](http://159.65.160.18:3020/)
Make sure you log in to see all the features, you can do so with the credentials 'carmen', pwd '123' or using the bottom right control panel

## Features

* Public recipes
* Recipe management 
* User auth
* Comments and favorites
* Emulated backend using local storage

## Tech stack

* ReactJS
* Vite
* Node.js / Express
* PM2
